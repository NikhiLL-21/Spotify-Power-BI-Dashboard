# 🎧 Spotify Data Analysis Dashboard

## 📌 Project Overview

This project presents a **Spotify Analytics Dashboard** designed to help stakeholders such as **music analysts, playlist managers, and marketing teams** monitor and analyze song and artist performance.

The dashboard transforms raw Spotify "Top 50" data into **interactive insights** for better decision-making.

---

## 🎯 Business Requirement

The dashboard is divided into three key sections:

### 📊 1. Overview Page

* Track key KPIs:

  * Total Songs
  * Distinct Artists
  * Average Popularity
  * Average Duration
* Compare **Explicit vs Non-Explicit Songs**
* Analyze songs by **Album Type** (Single, Album, Compilation)
* View **Distinct Songs & Avg Popularity by Year**
* Monthly trend analysis of:

  * Average Popularity
  * Distinct Songs
* Highlight:

  * 🔝 Top Songs
  * 🔝 Top Artists

---

### 🎤 2. Artist Page

* Identify **Top Artists by Popularity**
* Compare:

  * Tracks per Album
  * Songs by Artist
* Drill-down capabilities for:

  * Songs
  * Release Date
  * Average Popularity
  * Average Position
  * Duration
* Detect artists with:

  * Consistent hits
  * #1 ranked tracks

---

### 🎶 3. Songs Page

* Rank **Top Songs by Popularity**
* Analyze **Tracks per Song** (Album vs Single distribution)
* Compare songs based on **Song Count**
* Detailed table including:

  * Song Name
  * Release Date
  * Distinct Artists
  * Average Popularity
  * Position
  * Duration per Year

---

## ❗ Problem Statement

Spotify’s raw **Top 50 dataset** is limited to rankings and lists, making it difficult to extract meaningful insights quickly.

### Key Challenges:

* ❌ No centralized KPI monitoring
* ❌ Limited insight into explicit vs non-explicit content
* ❌ Poor visibility into album/song distribution
* ❌ Missing trend analysis over time
* ❌ No connection between artist-level and song-level insights
* ❌ Gaps in data-driven decision-making

---

## ✅ Solution

This dashboard solves the above challenges by:

* ✔️ Providing a **centralized KPI overview**
* ✔️ Enabling **explicit vs non-explicit comparison**
* ✔️ Visualizing **album type and yearly distribution**
* ✔️ Showing **monthly & yearly trends**
* ✔️ Offering **interactive drill-downs (Artist & Song level)**
* ✔️ Supporting **data-driven decisions** for marketing and playlist curation

---

## 🚀 Key Insights Enabled

* Identify **top-performing artists and songs**
* Discover **trending music patterns**
* Understand **content preferences (explicit vs non-explicit)**
* Optimize **marketing and playlist strategies**
